## Tecnologias Utilizadas

* React Native
* Expo
* Expo Router
* TypeScript

## Como Executar

1. Clone o repositório

```bash
git clone https://github.com/Alana125/sabores-da-rota-pmovel_Alana_e_Lyvia.git
```

2. Instale as dependências

```bash
npm install
```

3. Execute o projeto

```bash
npm run start
```

4. Abra o aplicativo no Expo Go.
